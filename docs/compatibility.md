
## 已知限制

1. **DRM 内容**（Netflix、Disney+ 等）无法投屏。
2. **需 Cookie 鉴权的 HLS** 电视无法直接拉流，v1 不支持。
3. **blob: URL** 无法投屏，需可直链访问的 HTTP(S) URL。
4. 部分电视仅支持特定容器（MP4/H.264 兼容性最好）。

## 电视端设置指引

### 小米电视

1. 设置 → 连接 → 无线投屏 / 投屏 → 开启
2. 确保与电脑在同一 Wi-Fi（避免访客网络隔离）
3. 在麻薯投屏 Popup 中点击「扫描」，或手动输入电视 IP（设置 → 网络 → 网络信息）

### TCL 电视

1. 设置 → 网络 → 多屏互动 / DLNA → 开启
2. 扫描或手动添加 IP

### 通用 DLNA 电视

1. 在电视网络设置中查看 IP 地址
2. 在麻薯投屏中使用「添加 IP」
3. 使用可直链的 MP4 视频验证
